HTML: Doctypes


The < !DOCTYPE html > declaration is required to specify the HTML version and rendering instructions and should be included at the top of an HTML document before the < html > tag. HTML5 doesn't need a DTD because it isn't based on SGML.

HTML 5 Doctype declaration: < !DOCTYPE html > Case Insensitivity the following DOCTYPES are also valid:
< !doctype html >
< !dOCtyPe html >
< !DocTYpe html >
